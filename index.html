8-bit Top-Down Demo

This prototype demonstrates the basic feel of the game: a top-down 8-bit view where the player can walk around a compact level. Obstacles block the character, letting you test collision, camera, and movement speed.

Input handling, collision, and the fixed-step loop now run on a local Phaser 3 build bundled with the page, so it works even if the CDN is blocked. The demo level also includes simple pickups and a six-slot inventory UI to test item collection.

  • WASD or Arrow Keys to move
  • Bump into walls to feel collision
  • Walk over glowing diamonds to collect items
  • Escape closes this overlay
